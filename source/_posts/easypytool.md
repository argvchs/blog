[EasyPyTool](https://github.com/argvchs/easypytool) 是一个简单的 Python 环境配置工具

<!-- more -->

## 文件

-   `config.sh`

    设置 Python 镜像源和代理

-   `export.sh`

    导出 PIP 库到 `packages`

-   `import.sh`

    导入 PIP 库

-   `setup.sh`

    下载 Python 安装文件

-   `clear.sh`

    清理导出的 PIP 包和安装文件
